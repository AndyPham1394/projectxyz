About this page

This it a personal project made on Nodejs. It's buggy and not fully functional, but also a good start. Here you can find some of the features implemented:

  • A Camera controller use to control remote camera module
  • A Realtime chat app build on nodejs, express and websocket APIs
  • Simple hardware used to run Local Server with Pi SBC
  • My poor frontend skills

About Machine controller

Camera controller allowed us to control remote camera module. Streaming and controls are available with websocket and html requests. Here are some informations:

solftwares:

  • HTTP server as transporter
  • Low delay commands through Websocket

hardwares:

  • 1 arduino uno module
  • 1 esp32cam module
