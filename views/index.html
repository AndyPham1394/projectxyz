About this page

This is a personal project made on Nodejs. It's buggy and not fully functional, but also a good start. Here you can find some of the features implemented:

  • Machine Manager used to connect to remote modules
  • A Realtime chat app build on nodejs, express and websocket APIs
  • Local Server with Pi SBC
  • My poor frontend skills

About Machine Manager

Connect with remote machines to control and get data from them through websocket connections and html requests.

hardwares:

  • 1 arduino uno module
  • 1 esp8266 module
  • 1 esp32cam module
